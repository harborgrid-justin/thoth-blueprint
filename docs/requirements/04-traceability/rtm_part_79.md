# Requirements Traceability Matrix - Part 79
**Subject:** Wall Cleanup Groups & Endcaps (Chapter 20, Part 2)
**Coverage:** Cleanup Groups, Dynamic Cleanup Radius, Endcaps Overrides, T-junction Intersections, Wall Component Wrapping

| Req ID | Tutorial Section / Reference | Requirement Description | Traceability | Code Mapping |
|---|---|---|---|---|
| REQ-79-001 | Chapter 20 > Walls > Cleanup Groups | The system shall support Wall Cleanup Groups specifying which walls automatically merge at intersections. | Trace-to-Spec-v1 | Not implemented |
| REQ-79-002 | Chapter 20 > Walls > Cleanup Radius | The system shall support setting a cleanup radius for walls intersections. | Trace-to-Spec-v1 | Not implemented |
| REQ-79-003 | Chapter 20 > Walls > Endcaps Overrides | The system shall support overriding wall endcap geometry at individual wall ends. | Trace-to-Spec-v1 | Not implemented |
| REQ-79-004 | Chapter 20 > Walls > T-junctions | The system shall automatically clean up T-junction wall intersections when in the same cleanup group. | Trace-to-Spec-v1 | Not implemented |
| REQ-79-005 | Chapter 20 > Walls > Component Wrapping | The system shall support wrapping specific wall components (e.g. wall cladding) at endcaps. | Trace-to-Spec-v1 | Not implemented |
