| REQ-10-004 | Alignments > Design Speed | The system shall allow specifying Starting Design Speed and station-specific design speeds along an alignment. | Trace-to-Spec-v1 | Specified in horizontal alignments `designSpeed` property in [alignment.ts](../packages/domain/src/alignment.ts) and configured in the wizard panel [SuperelevationWizardDialog.tsx](../apps/web/src/features/survey/SuperelevationWizardDialog.tsx) |

| REQ-10-012 | Alignments > Default Curve Settings | The system shall support configuring default curve insertion settings (type, radius, spiral type) for automatic curve placement. | Trace-to-Spec-v1 | Alignment tool automatically applies a default curve radius during canvas drawing in [PlanningCanvas.tsx](../apps/web/src/features/canvas/PlanningCanvas.tsx#L289) |
| REQ-10-013 | Alignments > From Xref Objects | The system shall support creating alignments from objects in externally referenced drawings using the Xref option. | Trace-to-Spec-v1 | Not implemented |
