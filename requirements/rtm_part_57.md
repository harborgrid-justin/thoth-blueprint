# Requirements Traceability Matrix - Part 57
**Subject:** Windows (Chapter 25)
**Coverage:** Window Styles (Double-Hung/Awning/Hopper), Custom Muntins, Wall Anchors, Glazing & Sash Display, Arch Shapes

| Req ID | Tutorial Section / Reference | Requirement Description | Traceability | Code Mapping |
|---|---|---|---|---|
| REQ-57-001 | Chapter 25 > Windows > Styles | The system shall support Window Styles defining standard double-hung, glider, awning, hopper, and fixed types. | Trace-to-Spec-v1 | Not implemented |
| REQ-57-002 | Chapter 25 > Windows > Muntins | The system shall support custom muntins division configurations (vertical/horizontal counts) inside window panels. | Trace-to-Spec-v1 | Not implemented |
| REQ-57-003 | Chapter 25 > Windows > Anchors | The system shall support anchoring windows to wall objects with automatic openings creation. | Trace-to-Spec-v1 | Not implemented |
| REQ-57-004 | Chapter 25 > Windows > Display | The system shall display window sashes and glass panes in 2D plan and 3D model viewports. | Trace-to-Spec-v1 | Rendered in 3D frame layouts in [buildScene.ts](../apps/web/src/features/canvas3d/buildScene.ts) |
| REQ-57-005 | Chapter 25 > Windows > Shapes | The system shall support rounded, arched, and triangular window geometry profiles. | Trace-to-Spec-v1 | Not implemented |
